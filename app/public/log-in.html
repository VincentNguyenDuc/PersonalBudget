ABOUT

Personal Budget is a practice Web Development project. In this project, I build an API using JavaScript, Node.js and Express.js. Clients will follow the Envelopes Budgeting Principles and be able to create, read, update, and delete their personal envelope. The front-end side is still under development, clients can access the features through endpoints.
